# Personal Portfolio Website

This project is my personal portfolio website. It showcases my projects, skills, and resume in a simple and clean layout. 
## Features
- Home section with my name and intro  
- Projects section showing my work in a grid layout  
- Resume button to view my PDF  
- Fully responsive design  
- Built with simple HTML, CSS, and JavaScript  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript
