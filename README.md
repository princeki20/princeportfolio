# Personal Portfolio Website

This project is my personal portfolio website. It showcases my projects, skills, and resume in a simple and clean layout. The site includes a dark “hacker mode” theme and a light mode, which can be toggled with a switch.

## Features
- Home section with my name and intro  
- Projects section showing my work in a grid layout  
- Resume button to view my PDF  
- Theme toggle (Hacker Mode ↔ Light Mode)  
- Fully responsive design  
- Built with simple HTML, CSS, and JavaScript  

## Technologies Used
- HTML5  
- CSS3  
- JavaScript
